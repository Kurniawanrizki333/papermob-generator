Papermob

Gambar → grid kode warna → Excel

Unggah gambar, tentukan ukuran grid, dan Papermob akan memetakan tiap pixel ke kode warna terdekat lalu menyusunnya jadi file Excel lengkap dengan sheet visual per gambar dan sheet daftar pixel gabungan. Semua proses berjalan di browser kamu — gambar tidak dikirim ke server manapun.

01 Ukuran grid

Mode Teks / poster menjaga huruf, border, dan ornamen tipis dengan sampling mayoritas warna. Pakai Pixel art kalau gambar memang sudah dibuat persis sesuai grid.
Grid besar ( > 150 × 150 ) bisa memperlambat proses di browser, terutama untuk banyak gambar sekaligus.

02 Gambar

Tarik & taruh gambar / PDF di sini, atau klik untuk pilih file
Mendukung PNG, JPG, WEBP, PDF · bisa pilih banyak file sekaligus · tiap halaman PDF jadi frame terpisah
Belum ada gambar diunggah.

03 Legenda kode warna

Aturan akhiran: akhiran -B (berdiri) diberikan untuk pixel yang menjadi bagian objek utama / area yang dihighlight, sedangkan -D (dasar) untuk pixel background polos. Ini berlaku untuk semua warna — termasuk putih: kalau putih tersebut bagian dari objek utama, tetap PT-B, bukan otomatis PT-D. Contoh: putih di background → PT-D, putih di objek utama → PT-B, merah → MR-B.
